# Blinkit Sales & Delivery Analytics Dashboard

An interactive **Power BI dashboard** built to analyze Blinkit's sales performance, outlet distribution, and product-level metrics. The project demonstrates end-to-end data analysis — from raw data cleaning to business insight visualization.

## Objective

To analyze Blinkit's grocery sales data and uncover insights around:
- Which product categories drive the most revenue
- How outlet size and location type affect sales performance
- Customer satisfaction trends across item types
- Fat content preferences across different outlet tiers

---

## Tools & Technologies

| Tool | Usage |
|------|-------|
| Power BI Desktop | Dashboard design & visualization |
| DAX | Custom measures and KPI calculations |
| Power Query | Data cleaning and transformation |
| Excel / CSV | Source data |

---

## Key KPIs Tracked

- **Total Sales** — Overall revenue generated
- **Average Sales** — Revenue per transaction
- **Number of Items** — Total product count
- **Average Rating** — Customer satisfaction score

---

## Dashboard Features

- **Dynamic slicers** — Filter by outlet size, location type, and item type
- **Donut charts** — Fat content breakdown (Low Fat vs Regular)
- **Bar charts** — Sales by item category (Fruits, Snacks, Dairy, etc.)
- **Line chart** — Outlet establishment trend over years
- **Funnel / stacked charts** — Outlet size and tier comparison
- **Matrix view** — Outlet type vs all KPIs side-by-side

---

## Key Insights

> *(Replace these with the actual numbers from your dashboard)*

- Supermarket Type 1 outlets contributed the highest share of total sales
- Low-fat products outsold regular-fat items across most outlet tiers
- Tier 3 locations showed stronger sales volume than Tier 1 in several categories
- Fruits & Vegetables and Snack Foods were the top-performing item types
